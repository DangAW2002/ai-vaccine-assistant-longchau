📈 High Consultation Volume
70% of incoming calls are variations of the same 5-10 questions, resulting in:
  • Large consultation team required (expensive)
  • Extensive training on vaccination protocols
  • Difficult to scale during peak times
  • Employee burnout due to repetitive work
💰 Cost Impact
Current hotline operations cost ~50 million VNĐ/year. AI automation estimated to reduce by 60%.

MVP Scope

Single User Journey - Happy Path:

  1. 25-year-old seeking HPV vaccine information
  2. AI asks clarifying questions (age, gender, health status)
  3. AI analyzes and recommends vaccination schedule
  4. User searches for nearby clinics
  5. User books appointment online
  6. Confirmation sent via email/SMS

Design Principles

Principle Implementation
Safety First Escalate when uncertain; never override professional medical judgment
Transparency Always show reasoning; explain "Why this vaccine?"
User Control Users can reject recommendation or ask alternatives
Graceful Fallback Seamless handoff to human agents
24/7 Availability Instant response, no wait time

AI Agent Prompt Strategy

Few-Shot Prompting Approach:
  • System Prompt: Role as vaccination expert, Vietnamese healthcare context
  • Few Shots: 3-5 example conversations showing desired behavior
  • Context: Official Vietnamese immunization schedule
  • Guardrails: Restrictions on medical advice for complex cases

Sample Prompt

You are a helpful vaccination consultant at Long Châu pharmacy in Vietnam. Your role is to help customers understand which vaccines they need. IMPORTANT RULES: 1. Always ask for age, gender, and health conditions before recommending 2. Only recommend vaccines from the standard Vietnamese immunization schedule 3. If the case is complex (chronic disease, allergies), escalate to pharmacist 4. Always explain WHY a vaccine is recommended 5. Show confidence level (certain/moderate/uncertain) 6. If uncertain, say "Tôi không chắc" and offer to connect with pharmacist VACCINATION DATABASE: - HPV (Gardasil 9): Ages 9-45, 3 doses, 6-12 month schedule - Flu (Seasonal): Everyone annually, 350-500k VNĐ - Pneumococcal (Prevenar 13): Adults 65+, 1 dose, 650k VNĐ ... [more vaccines] USER: Tôi 32 tuổi, nữ, chưa tiêm HPV bao giờ ASSISTANT: Rất tốt! Ở độ tuổi 32, bạn hoàn toàn có thể tiêm vắc xin HPV. - **Vắc xin khuyến cáo:** Gardasil 9 (bảo vệ 9 loại HPV) - **Số mũi:** 3 mũi - **Lịch tiêm:** Mũi 1 → Mũi 2 (2 tháng sau) → Mũi 3 (6 tháng sau) - **Chi phí:** ~2.4 triệu VNĐ (3 mũi × 800k) - **Độ tin cậy:** 🟢 CERTAIN - Đây là quy tắc chuẩn Bạn sẵn sàng tiêm không?

Error Handling

Scenario Detection Recovery Strategy
AI suggests non-existent vaccine Validate against vaccine DB Fallback + escalate
Missing critical patient info Pydantic validation Re-ask specific question
Chronic disease detected Keyword detection Mandatory escalation
API timeout (OpenAI) Request timeout >5s Use cached response or escalate
Clinic search fails Empty results Show alternative options or hotline
